## SECTION 1: USER TYPES & CAPABILITIES

### **1. CONTRACTOR** (Trade Professional)
**Definition:** Licensed/unlicensed skilled worker (electrician, plumber, carpenter, etc.)

**Core Capabilities:**
- Browse & bid on open jobs
- Create scope of work proposals
- Accept/complete jobs
- Invoice & receive payouts
- Build portfolio (before/after photos)
- Receive reviews & ratings
- Use project collaboration tools (chat, milestones, files)

**Communication:**
- **With Clients:** Via project chat (scoped to specific job)
- **With Platform:** Admin-only messaging available
- **With Other Contractors:** None (isolated profiles)

**Without Optional Products:**
- Basic job browsing
- Simple scope submission
- Manual invoice creation
- Standard 18% platform fee on payments

**With Optional Products:**
- **WAVE FO (Field Operations):** Real-time job management, GPS tracking, expense logging, crew coordination, report generation
- **WAVE Residential Bundle:** Invoicing + document templates + business tools
- **WAVE Pro/Max/Premium:** Enhanced features + priority support

**Restrictions & Why:**
- 18% platform facilitation fee (payment processing, support, infrastructure)
- Cannot message clients directly until job accepted (fraud prevention)
- After-photo deadline: 72 hours post-completion (quality assurance)
- Account lock if after-photos missed (ensures accountability)
- Payment held in escrow until client approves work (protects both parties)

**Pricing (Optional/Upgrades):**
- **WAVE FO Premium:** $100/month (full field operations suite)
- **WAVE Starter:** $19/month (basic tools)
- **WAVE Pro:** $39/month (advanced features)
- **WAVE Max:** $59/month (max features)
- **Residential Bundle:** $125/month (complete business package)

**Tiers Unlocked By Job Completion:**
- **Wave** (0 jobs): Standard tier
- **Ripple** (15-34 jobs): Unlock WAVE FO Lite, AI assistant
- **Swell** (35-54 jobs): Full WAVE FO access
- **SurfCoast WAVE FO** (55+ jobs): Premium field ops with advanced analytics

---

### **2. CUSTOMER / CLIENT** (Homeowner/Property Owner)
**Definition:** Individual who posts jobs and hires contractors.

**Core Capabilities:**
- Post job descriptions with budgets
- Upload before photos
- Review & approve contractor proposals
- Chat with contractors (once proposal selected)
- Approve completed work
- Pay via Stripe (escrow-protected)
- Leave reviews/ratings

**Communication:**
- **With Contractors:** Scoped project chat (message-per-job fee: $1.50)
- **With Customer Support:** Via admin contact form
- **With Other Customers:** None

**Without Optional Products:**
- Unlimited job posting
- Free scope reviews from contractors
- Standard escrow protection
- 30-day payment window

**With Optional Products:**
- **Quote Request Wizard:** $29.99 (get multiple proposals at once, priority responses)
- **Residential WAVE Bundle:** $125/month (document library, proposal templates, payment tracking)

**Restrictions & Why:**
- Must provide 5+ before photos (ensures clear scope definition)
- Message fee ($1.50/hr session) funds contractor-client communication (prevents spam)
- 14-day approval window post-completion (encourages timely closeout)
- If job unpaid after 14 days → dispute escalation (cash flow protection)
- Escrow release delay (payment held pending review) (fraud prevention)

**Pricing:**
- **Quote Request Wizard:** $29.99 one-time (optional, unlocks batch proposals)
- **Residential WAVE Bundle:** $125/month (optional, includes invoicing templates, document management)
- **Limited Comm Sessions:** $1.50 each (mandatory to message contractors)

**Tiers Unlocked:**
- **Consumer** (0 jobs posted): Standard tier
- **HomeBuilder** (5-15 jobs): Badge + priority contractor matching
- **BuildMaster** (16+ jobs): Badge + exclusive contractor network

### **3. MARKET VENDOR** (Reseller/Service Provider)
**Definition:** Small business/individual selling products/services at farmers markets, swap meets, etc.

**Core Capabilities:**
- Create market shop (product listings)
- Manage inventory
- Accept consumer orders
- Communicate with buyers
- Track sales analytics
- Process payments via Stripe
- Set up booth at events

**Communication:**
- **With Consumers:** Direct messaging (no fee)
- **With Other Vendors:** Event coordination only
- **With Platform:** Admin alerts for low stock, payment issues

**Without Optional Products:**
- Basic shop setup
- Product listings
- Standard 20% platform fee on sales
- Manual inventory tracking

**With Optional Products:**
- **Market Shop Subscription:** $20/month (enhanced analytics, bulk messaging, campaign tools)
- **WAVEshop FO:** $35/month (full inventory management, crew coordination, event planning)

**Restrictions & Why:**
- 20% platform fee on sales (payment processing, dispute handling, platform infrastructure)
- Must maintain inventory accuracy (prevents overselling)
- Shop cannot be deleted while active orders exist (order fulfillment protection)

**Pricing:**
- **Market Shop Subscription:** $20/month (optional, analytics + marketing)
- **WAVEshop FO:** $35/month (optional, full operations suite)
- **Event Booth Rental:** Varies by market ($50-200/event, paid to market organizer)

**Tiers Unlocked:**
- **Vendor** (0-10 sales): Standard tier
- **Trusted Vendor** (11-50 sales): Badge + featured listing eligibility
- **Premier Vendor** (50+ sales): Badge + priority event placement

## SECTION 8: COMMUNICATION RESTRICTIONS EXPLAINED

**Why $1.50 Messaging Fee?**
- Prevents spam (bots, scams, unsolicited offers)
- Funds contractor support (moderation, dispute handling)
- Ensures serious buyer-contractor interactions only
- Alternative: Proposal-based communication (free once job accepted)

**Why 72-Hour Photo Deadline?**
- Ensures fresh evidence of work quality
- Prevents fake/stock photos
- Protects client (knows exactly what they got)
- Protects contractor (photos timestamped)

**Why Escrow Hold Until Approval?**
- Client verifies work before contractor gets paid
- Contractor knows work accepted before releasing funds
- Dispute resolution mechanism (escrow held during disputes)
- Both parties incentivized to complete quality work
